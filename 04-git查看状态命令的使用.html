1. 查看状态 git status 查看简要状态 git status -s 1. 添加状态 (红色) Untracked files: (use "git add ..." to include in what will be committed) 有文件未添加（没有添加和提交到本地仓库） 2. 修改状态 (红色) modified: index.html 表示有文件修改了但是还没添加和提交到本地仓库 3. 添加了未提交（绿色） Changes to be committed: (use "git reset HEAD ..." to unstage) new file: demo.html modified: index.html 4. 没有任何修改和提交 (纯白色) (use "git push" to publish your local commits) nothing to commit, working tree clean 5. 删除了但是未添加和提交（红） Changes not staged for commit: (use "git add/rm ..." to update what will be committed) (use "git checkout -- ..." to discard changes in working directory) deleted: cart.html 6. 删了并且添加 但是未提交 （绿色） (use "git push" to publish your local commits) Changes to be committed: (use "git reset HEAD ..." to unstage) deleted: cart.html
